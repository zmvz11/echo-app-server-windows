# Final Build Checklist

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run final-check`
- Create first owner
- Register a normal user
- Approve pending user
- Create an app
- Upload icon/banner/screenshot
- Upload a zip release package
- Submit, approve, and publish the release
- Confirm `/api/catalog` includes the published app and release
- Confirm App Center can install and launch the app through the local agent
